# 肌肉图谱 3D · Muscle Atlas 3D

交互式健身肌肉解剖工具。基于真实解剖 3D 模型，帮助你直观看到**每个健身动作练哪些肌肉、动作如何牵动肌肉收缩**，并提供**肩部疼痛的结构定位**（教育参考）。

## 功能

**动作解剖**
- 真实人体肌肉 3D 模型，可拖动旋转、缩放。
- 选择 26 个常见健身房动作，参与肌肉按角色高亮：**主动肌（红）/ 协同肌（橙）/ 稳定肌（金）**。
- 发力肌肉做**真实几何收缩形变**（沿肌肉长轴缩短、增粗）+ 发光脉冲，配向心/离心相位与呼吸提示。
- 相机自动聚焦到对应部位（正/背面、上/下半身）。
- 点击任意肌肉 → 反查所有练它的动作。
- 可切换显示骨骼、深/浅色主题。

**肩痛定位**（仅供教育参考，非医疗诊断）
- 点击肩部结构（肩峰下、冈上肌、冈下/小圆、二头长头腱、肩锁关节、肩胛内缘、三角肌后等）。
- 深层结构自动透视，显示由浅入深的解剖层次、常见对应问题、简易自测、训练建议与红旗警示。

## 本地运行

模型和三方库全部本地打包，无需联网、无 CDN。用任意静态服务器打开：

```bash
python -m http.server 8000
# 浏览器打开 http://localhost:8000
```

> 注意：因浏览器安全策略，需通过 http(s) 服务器打开，直接双击 `file://` 打开无法加载模型。

## 技术

纯前端：Three.js（本地 ES module）+ DRACO 压缩的 `body.glb`（约 8MB）。肌肉按网格节点名映射到动作库；收缩动画为每帧对活动肌肉网格顶点做程序化形变。
